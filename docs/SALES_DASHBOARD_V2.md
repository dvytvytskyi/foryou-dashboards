# 🚀 Нова Структура Дашборду: Відділ Продажу (V2)

Цей документ описує оновлену концепцію дашборду, яка фокусується на бізнес-показниках та ефективності, згідно з новими вимогами.

---

### 📋 Основні Блоки Дашборду

| Блок | Що бачимо (KPI) | Основна цінність |
| :--- | :--- | :--- |
| **1. Загальні Цифри** | Дохід, к-ть угод, сер. чек, конверсія. | **Швидкий погляд:** Як працює відділ прямо зараз. |
| **2. Деталі Напрямків** | Розподіл: Off-plan, Оренда, Вторинка. | **Фокус:** Який ринок приносить найбільше грошей. |
| **3. План / Факт** | % виконання плану відділу та кожного брокера. | **Дисципліна:** Хто встигає до цілі, а хто ні. |
| **4. Брокер у деталях** | Угоди, дохід, якість роботи (касання). | **Аудит:** Персональна ефективність кожного агента. |
| **5. Аналітика Лідів** | Джерела (FB, Insta, Google), ROI (Витрати vs Дохід). | **Маркетинг:** Куди вигідно вкладати гроші. |

---

### 🛠 Деталізація Кожного Блоку

#### 🟢 1. Загальні цифри (Lead Audit & SLA)
*Головний екран контролю якості обробки лідів.*
- **Lead Audit:** Загальна к-ть лідів у CRM, к-ть "втрачених / нереалізованих / недозвонів".
- **Rule of 6:** По кожній категорії — скільки було зроблено 6+ дотиків (таблиця).
- **SLA по брокеру:** Швидкість реакції (отримання заявки -> перший дотик). WhatsApp vs Дзвінок.
- **Причини відмови:** Аналіз "Чому лід втрачено?" (Дорого / Недозвон / Спам).
- **Якість дзвінків:** Окремий аудит дзвінків довше 30 секунд.
- **Гнучкість:** Фільтри за датою, брокером та джерелом.

#### 🔵 2. Деталі по напрямках
*Розуміння структури бізнесу.*
- **Off-plan:** Кількість та сума.
- **Оренда:** Стабільний потік.
- **Вторинка:** Рентабельність.
- **Додатково:** Порівняльний графік прибутковості кожного напрямку.

#### 🟡 3. План / Факт (Командний результат)
*Контроль виконання цілей.*
- **Відділ продажів:** Загальна плашка прогресу (н-р, 85% плану виконано).
- **Брокер загальний:** Зведений KPI по всіх агентах (Power Ranking).

#### 🟠 4. Брокер у деталях
*Глибоке занурення в роботу агента.*
- **Воронка:** Скільки лідів взяв -> скільки зустрічей -> скільки закрив.
- **Якість:** Дотримання регламенту (правило 6+ касань).
- **Гроші:** Скільки чистого прибутку приніс компанії.

#### 🔴 5. Деталі по лідам (ROI & Джерела)
*Зв'язок маркетингу та продажів.*
- **Джерела:** Звідки прийшли ліди? (Instagram, Web, Referral).
- **Ефективність:** Яке джерело дало "найнайдорожчі" угоди.
- **Економіка:** Скільки витратили на рекламу vs Скільки отримали доходу (ROI).
