# Scratch Directory

This directory is for temporary experiments, legacy extraction helpers, and one-off utilities.

## Important

- Do not treat files in this directory as canonical production workflows.
- Do not use this directory as the first place to search for application behavior.
- If a script becomes part of a recurring operational flow, move it into `scripts/`.

## Intended Use

- temporary data extraction
- manual investigation
- migration experiments
- archived operational snippets
